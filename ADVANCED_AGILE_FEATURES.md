## 🚀 **Getting Started**

### **Quick Setup**
1. **Backend Setup**: 
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Frontend Setup**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Access the Application**:
   - Frontend: http://localhost:5180
   - Backend API: http://localhost:4000

### **Feature Access**
Navigate to any project and access the advanced workflow features through the comprehensive tab navigation:
